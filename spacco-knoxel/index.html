Knoxel: Coding voxels in your browser!

Background

Minecraft is a popular voxel-based game played by millions around the world. Knoxel brings the joy of Minecraft to CS0/CS1!

With Knoxel you write code in JavaScript, Python, Ruby, or Java to manipulate 3D arrays of blocks, then visualize your creations in a voxel-based world, right there in your web browser!

Knoxel is free, with no software to install for the faculty or the student!

We make everything happen in your browser through the magic of Javascript and a healthy amount of cursing!

Meta

Summary Students write code in Javascript, Python, Ruby, or Java to manipulate 3D arrays of blocks. We provide enums or enum-like constants for each language to make this process simple. Students can then render and view their creations in their web browser!
Topics 3D arrays, nested for loops, 3D volumes.
Audience CS0 or CS1.
Difficulty

I usually give them some sample programs and a week to craft their drawings.

The instructions are usually something like, "Make a house! A house should have a roof and a door and a hollow space inside." They can interpret what "house" means in a variety of different, interesting ways.

Strengths
  • Minecraft remains incredibly popular
  • Requires no software installation by students or the instructor!
  • Great for open-ended, creative assignments where "correct" answers look very different
  • Student creations make for good demos or CS0/CS1 hall-of-fame displays
Weaknesses
  • Compiler and runtime error messages through Knoxel are different than what students see with their IDE.
  • Students cannot save code written into the browser, so they might lose their work!
  • Some students do not like open-ended assignments lacking a single right answer.
  • We have not gotten Java support working in the browser, so they need to run it offline and upload a file it produces. This is how I use the system in CS1, and it's not a problem.
  • Lava, water, and other things that "flow" in regular Minecraft do not actually flow in Knoxel
  • You can't actually do any of the things you can do in Minecraft, like mine or place blocks, except by writing code. Faculty tend to like this feature, students may not.
Dependencies 3D Arrays, nested for loops.
Variants
  • Make parameterized structures, like "a function to create a tower where the height works for any multiple of 4".
  • Make specific structures.
  • Ask students to create the utility functions that create a 3D array of null with given dimensions, or to convert a 3D array into JSON.
  • The JSON files describing the blocks in a 3D array might be a good way to talk about data exchange formats, but I have not tried that yet.
